# Multibranch Jenkins Pipeline Demo.

<h5><b>Jenkinsfile:</h5></b>

  This jenkins file will build and push an nginx docker image. Different stages are executed based on branches and pull requests. PollSCM is used as automated trigger.
  
<h5>Dockerfile:</h5>

  Its a simple dockerfile to create an nginx image from ubuntu-16.04.
